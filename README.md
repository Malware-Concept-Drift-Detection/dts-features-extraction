## Getting Started

### Installation

1. Clone the repository:

   ```bash

   cd your-repository
   
2. Create the virtualenv:

    ```bash
   python3 -m venv .venv
   source .venv/bin/activate
   
3. Install the dependencies
    ```bash
   pip install -r requirements.txt
